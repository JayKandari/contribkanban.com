About Contrib Kanban

What's this built out of?

Shoe strings and tree sap.

Actually, it's an AngularJS app tapping into the Drupal.org REST APIs. We're then harnessing Parse for data storage.

Adding a project

Scraping Drupal.org and adding each project would be.. well expensive and timely. If you don't see your project, simply enter the machine name into the form on the homepage and add it.

Board lists

Currently the kanban board lists are set to the Drupal.org issue states. In the future there will be support for project maintainers to customize these lists to reflect issue tags and control sprint boards/

  • Postoned: Postponed & Needs more info
  • Active
  • Needs Review
  • Needs Work
  • RTBC
  • Fixed: Currently just Fixed, no Closed (fixed) for now.
  • Won't Fix Hidden, due to possible performance/number of calls. Anything closed and not fixed.
